# BIO-482-project

## Getting Started

Pour Neila et Angelina, placez vous où vous voulez sur votre ordi et ouvrez une fenêtre du temrinal. 

```
git clone https://github.com/notLuluu/BIO-482-project.git
```

Ensuite 
```
cd BIO-482-project
```

Vérifier que tout est bien là et ajouter le dossier ```Data``` que le prof a fait dans son dossier à lui. Le dossier ```Data``` est trop gros pour être posé sur github, vous l'avet donc localement sur votre ordi. Quand vous pushez votre travail, il ne s'ajoutera pas au github.
